# Polymarket Market Data Dashboard

GitHub User: panrui1984
ProjectName: Polymarket Market Data Dashboard
Track: 预测市场赛道
ProjectDescription: Polymarket Market Data Dashboard 是一个全栈预测市场数据分析平台。它不仅能实时索引 Polygon 链上的原始交易数据，还深度集成了 Google Gemini 2.0，将冰冷的十六进制链上日志转化为极具投资参考价值的智能情报。 本项目的核心目标是解决预测市场信息不对称问题，通过 AI 解码复杂的“负风险 (Negative Risk)”架构，识别巨鲸动向，并为每一个市场提供“情绪诊断”。
