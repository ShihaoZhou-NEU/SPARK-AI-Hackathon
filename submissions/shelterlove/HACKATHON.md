# PayPai

GitHub User: shelterlove
ProjectName: PayPai
Track: 支付赛道
ProjectDescription: 我们基于Kite链的Account Abstraction技术打造了⼀个会思考的智能钱包｡
核⼼能⼒⼀：AI⻛控⼤脑｡当你说"给A转100 USDT"， AI不仅⾃动识别地址，还会实时分析这个地址历史交易如何是否在⿊名单，当前Gas费是否合理，像银⾏的反欺诈系统，但完全去中⼼化､ 完全为你服务｡
核⼼能⼒⼆：可编程⽀付规则｡基于ClientAgentVault智能合约，⽤户可以设置"每⽇消费上限"､ "只允许⽩名单交易"､ "定期⾃动定投"等自定义交易规则｡⼀次配置后，可以用自然语言让AI⾃动执⾏，让资⾦管理从⼈⼯操作变成⾃动化｡
为什么选择Kite：Kite是专为AI优化的Layer 1，我们利⽤Kite的Paymaster机制实现零Gas体验，更重要的是Kite的低延迟和⾼吞吐让AI推理和链上执⾏可以实时协同这在其他链上很难实现｡
技术亮点 我们将Qwen⼤模型与链上智能合约深度整合，给AI Agent单独配备了Vault，共享AA Wallet，实现了意图识别→⻛险评估→交易⽣成→⾃动执⾏的全流程⾃动化｡ 代码已开源，合约已部署在Kite测试⽹，可以演示｡
