# Kite Drive

GitHub User: cheng-zhaofeng
ProjectName: Kite Drive
Track: 支付赛道
ProjectDescription: Kite‑Drive 是一个自动驾驶场景的支付项目，车在到达目的地前自动筛选车位、预定车位、利用 Kite AI 自动完成支付并规划泊车路线的全流程。前端以车载座舱 UI 形式展示“车辆状态 + AI 决策 + 交易过程”，后端负责车位决策、LLM 解释与 Kite AI 支付调用，整体强调“无感停车 + 自动支付”的全自动化体验闭环。解决了车辆高峰期找不到停车位，预约困难，支付困难的问题。项目在自动选定车位后，调用 Kite AI 支付后，可完成停车位锁定，无须担心其他车辆抢停等现象，保障了自动驾驶与自动支付的体验闭环。
